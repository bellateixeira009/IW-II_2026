Introdução ao DOM (JavaScript)

Compreender o que é o DOM e como o JavaScript interage com elementos HTML.

1. O que é DOM? Explique com suas palavras e a relação com HTML.

O DOM é como uma árvore genealógica dos elementos do documento para organizar a estrutura da paágina WEB.


2. O que é um nó (node)? Explique elementos, atributos e textos.

Se a estrutura do documento é uma árvore, os elementos, atributos e os textos são os galhos, os "nós" da árvore.

O elemento é a tag como p, div ou h1. Ele serve pra estruturar o conteúdo da página.

O atributo é uma informação extra que fica dentro da tag. Por exemplo, em p class="texto", para adicionar detalhes ou configurações ao elemento.

O texto é o conteúdo que fica dentro da tag, ou seja, aquilo que aparece para o usuário. Por exemplo, em

Olá mundo

, o "Olá mundo" é o texto.
